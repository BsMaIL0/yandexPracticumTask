Вам дана заготовка класса Person для хранения информации о человеке.

class Person {
    ...
private:
    std::string name_;
}; 
Что нужно сделать
Добавьте в этот класс:
Метод GetName, возвращающий имя.
Конструктор преобразования, принимающий параметр name. Запретите для конструктора неявные преобразования.
Пример объявления объектов класса:

// Ошибка: неявно создать объект нельзя.
// Person p1 = "Gwen"s;

// Создание объекта с явным преобразованием.
Person p2 = Person("Peter"s);
