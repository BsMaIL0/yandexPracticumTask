В уроке мы показали пример того, как использовать перегрузки присваиваний для записи данных одного типа в другой. Давайте используем это свойство, чтобы объект класса Rational можно было задать целым числом.
Что нужно сделать
Переопределите операцию = для Rational так, чтобы она принимала значение типа int и присваивала его дроби.
Добавьте конструктор с параметром int для аналогичного действия.
Добавьте операцию = для присваивания других дробей с реализацией по умолчанию.
Пример использования операции присваивания для целых чисел:

Rational r = 2; // Использовали конструктор:  r = 2 / 1.
r = 5;          // Использовали присваивание: r = 5 / 1.
    
// Можно использовать каскадирование.
Rational r_new = r = 10; // r_new = 10 / 1 и r = 10 / 1. 
В заготовке вам дан класс Rational с перегруженной операцией сравнения, которая поможет при проверке результатов.

Тест 1

Rational r = 2;
assert(r == Rational(2, 1));

r = 5;
assert(r == Rational(5, 1));

r = -9;

r = 0;
assert(r == Rational(0, 1));
