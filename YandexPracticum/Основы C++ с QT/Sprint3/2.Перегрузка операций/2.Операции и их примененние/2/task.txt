В этой задаче вам предстоит поработать с типом, похожим на Vector2D. Но определить его не как структуру, а как пару:

using IntPair = std::pair<int, int>; 
Что нужно сделать
Напишите следующие функции для работы с IntPair:

using IntPair = std::pair<int, int>;

// Складывает покомпонентно две пары.
IntPair Add(const IntPair& pair1, const IntPair& pair2);

// Вычитает покомпонентно две пары.
IntPair Sub(const IntPair& pair1, const IntPair& pair2);

// Умножает компоненты пары на число.
IntPair Mul(const IntPair& pair, int value);

// Печать и чтение интовой пары.
void Print(std::ostream& out, const IntPair& point);
IntPair Read(std::istream& inp); 
В заготовке дана функция main, которая читает выражение и выполняет запрошенную операцию.
Формат печати пары следующий: {A,B}, где вместо A и B — компоненты пары. Этот формат нужно использовать как в Print, так и в Read.
При считывании проверять корректность записи не обязательно. Вы можете прочитать любые символы между числами, рассчитывая на то, что подаётся корректный ввод.
Ограничения
В функции Print не нужно выводить перенос строки в принятый поток.
Примеры
Пример 1
Входные данные
{0,-15} + {1,2} 
Результат
{1,-13} 
Пример 2
Входные данные
{0,-15} - {1,2} 
Результат
{-1,-17} 
Пример 3
Входные данные
{0,-15} * 2 
Результат
{0,-30} 
Пример 4
Входные данные
{0,-15} / 2 
Результат
Undefined symbol! 
Код выхода: 1
