На складе Маркета находится множество товаров. Их артикулы в виде чисел int записываются в переменную типа std::vector<int>. Чтобы склад функционировал, нужно реализовать две операции:
Поставка. Она приходит в виде вектора артикулов. Новые артикулы нужно добавить к уже хранящимся на складе.
Списывание товаров. От поставщика приходит вектор артикулов, и эти товары нужно удалить со склада.
Система хранения склада задаётся объектом класса WarehouseStorage. Предположим, что переменная storage хранит объект этого класса, а sku_vector — объект типа vector<int>. Тогда, чтобы осуществить поставку, используется операция +:

// Для поставки прибавляем к складу вектор артикулов.
storage = storage + sku_vector; 
Списание будет реализовано через вычитание:

// Для списания вычитаем из склада вектор артикулов.
storage = storage - sku_vector; 
Что нужно сделать
Реализуйте методы добавления и списания товаров со склада (векторы чисел) для класса WarehouseStorage в виде операций.

class WarehouseStorage {
public:
  WarehouseStorage operator+(const std::vector<int>& vec) const;
  WarehouseStorage operator-(const std::vector<int>& vec) const;
} 
В заготовке дана функция main, которая прочитает два вектора и выполнит указанные операции.
