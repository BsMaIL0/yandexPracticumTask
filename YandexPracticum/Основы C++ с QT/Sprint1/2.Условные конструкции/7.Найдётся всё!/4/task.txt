Эта задача является финальной задачей темы, а также послужит фундаментом для финальной задачи следующей темы.
Что нужно сделать
Напишите программу, которая считает из потока:
A — число double,
арифметическую операцию,
B — второе число double.
Затем выведет результат вычисления прочитанного выражения.
Допустимы операции +, -, *, /.
После чтения проверьте поток — если какая-то операция чтения не удалась, выведите текст Incorrect input.
Если введена некорректная операция, выведите текст Incorrect operation.
Ошибки следует выводить в std::cerr, результат выражения в std::cout.
Как будет тестироваться код
Вывод вашей программы будет сравниваться с эталонным.
Примеры
Пример 1
Входные данные
2 + 2 
Результат
4 
Пример 2
Входные данные
3 * 3 
Результат
9 
Пример 3
Входные данные
-3 + 3 
Результат
0 
Пример 4
Входные данные
5 / 0 
Результат
inf 
Пример 5
Входные данные
-5 / 2 
Результат
-2.5 
Пример 6
Входные данные
My name is Guido 
Результат (std::cerr)
Incorrect input 
Пример 7
Входные данные
33.5 Guido 46.9 
Результат (std::cerr)
Incorrect operation
