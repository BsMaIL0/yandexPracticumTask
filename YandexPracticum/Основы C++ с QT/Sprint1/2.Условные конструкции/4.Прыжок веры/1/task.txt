Что нужно сделать
Задан enum, содержащий названия дней недели по порядку.
Напишите программу, которая считает целое число N из потока ввода — номер дня недели — и выведет название дня недели по его номеру. Если задан некорректный номер дня недели (меньше 1 или больше 7), программа должна вывести Нет такого дня недели. В этой задаче удобно использовать конструкцию switch-case по enum.
Ограничения
Используйте switch по значению enum-типа.
Как будет тестироваться код
Вывод вашей программы будет сравниваться с эталонным.
Также будет проверено ограничение.
Примеры
Пример 1
Входные данные
1 
Результат
Monday 
Объяснение
Программа вывела Monday, так как это день недели с номером 1.
Пример 2
Входные данные
4 
Результат
Thursday 
Пример 3
Входные данные
0 
Результат
Нет такого дня недели 
Объяснение
Программа вывела Нет такого дня недели, так как день недели с номером 0 отсутствует.
