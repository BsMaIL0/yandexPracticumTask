Что нужно сделать
Напишите программу, которая считает из потока ввода два числа int и выведет в поток наибольшее из них.
Программа должна вывести фразу Число X больше, в которой вместо X указано большее число. Если числа равны, то программа выводит фразу Числа равны.
Как будет тестироваться код
На вход будут подаваться целые числа N и M, укладывающиеся в int. 
Вывод вашей программы будет сравниваться с эталонным.
Примеры
Пример 1
Входные данные
5 3 
Результат
Число 5 больше 
Объяснение
Программа вывела 5, так как число 5 больше числа 3.
Пример 2
Входные данные
-3 11 
Результат
Число 11 больше 
Пример 3
Входные данные
42 42 
Результат
Числа равны
