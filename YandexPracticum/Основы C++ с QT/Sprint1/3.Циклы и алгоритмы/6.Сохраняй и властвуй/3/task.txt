В некоторых случаях для выполнения вычислений требуются большие целые числа, а иногда и очень большие. Однако для таких чисел не предусмотрены стандартные типы данных. Одним из способов их хранения является запись в виде вектора поразрядной величины.
Ещё одна сложность — для очень больших чисел нет стандартных арифметических операций. Поэтому эти операции приходится реализовывать самостоятельно.
Итак, задача: реализуйте операцию прибавления единицы для сколь угодно больших чисел.
Что нужно сделать
Необходимо реализовать код инкремента для очень большого целого числа.
Считайте из потока ввода число — строку, состоящую из цифр, — и прибавьте к этому числу единицу. Выведите полученное число.
Учтите, число может быть такой длины, что не поместится ни в int, ни в std::uint64_t. Считайте его как строку, а затем переведите цифры (отдельные символы этой строки) в элементы вектора.
Удобно использовать обратную запись: первая цифра числа — последний элемент вектора.
Помните, что прибавление к числу единицы может затронуть много разрядов. Тут пригодится цикл. Если единицу прибавить к числу из девяток, длина числа увеличится.
Алгоритм решения
Для начала прочитайте число и сохраните его в переменную типа string.
Разряды числа удобно хранить в векторе чисел — std::vector<int>.
Чтобы сохранить цифры в вектор std::vector<int> в обратном порядке, нужно пройтись по прочитанной строке справа налево и выполнить push_back в вектор на каждом этапе.
Не забывайте, что str[i] имеет тип char. Если это цифра, например '5', то преобразовать её в число можно с помощью конструкции: int digit = static_cast<int>(str[i] - '0').
Когда складывают в столбик, обычно приговаривают: «0 пишем, 1 в уме». Под «умом» в данном случае подразумевают память — у компьютера она есть. Поэтому «1» нужно сохранить в переменную. Подойдёт булева переменная, которая подскажет, был ли «1 в уме» на предыдущем разряде.
Можно пойти на хитрость: изначально инициализировать эту переменную в true. Тогда достаточно будет присвоить ей false в единственном случае — когда «держать в уме» больше не придётся.
Когда «1 в уме» больше не нужен, можно прервать цикл командой break.
Если после прохождения по всему числу остался «1 в уме» — он будет новым разрядом, который можно добавить в вектор методом push_back.
В конце распечатайте элементы вектора по одному в обратном порядке.
Схема программы может быть такой:
Объявление переменных.
Чтение строки.
Перенос цифр в вектор (цикл).
Прибавление единицы (цикл).
Добавление ещё одной единицы, если требуется (if).
Вывод элементов вектора (цикл).
