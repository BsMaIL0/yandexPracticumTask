Продолжаем работать с векторами. В этой задаче вам будет необходимо «прочитать вектор с конца». 
Что нужно сделать
Считайте из потока ввода число N и далее N строк и сохраните их в вектор. Считайте из потока ввода число К. Выведите К элементов вектора в формате <индекс> <строка>, начиная с последнего.
Для считывания строк целиком используйте функцию std::getline. Вспомните о приёме применения std::getline для перехода на следующую строку. Не забывайте и о защите от переполнения в циклах.
Как будет тестироваться код
Вывод вашей программы будет сравниваться с эталонным.
Примеры
Пример 1
Входные данные
5
C++ is the best
Python is good
Perl is pretty ugly
Fortran is so old
Cobol is done
2 
Результат
4 Cobol is done
3 Fortran is so old 
Объяснение
Программа вывела два последних элемента вектора в формате <индекс><пробел><строка>.
