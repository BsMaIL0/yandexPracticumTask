Числа Фибоначчи — это последовательность чисел, которая начинается с 0 и 1, а каждое последующее число — это сумма двух предыдущих: 0, 1, 1, 2, 3, 5, 8, 13, 21, … Мы будем нумеровать эти числа с нуля. Таким образом, 0 — нулевое число Фибоначчи, 1 — первое, 8 — шестое.
Эта последовательность была впервые рассмотрена итальянским математиком Леонардо Пизанским, она имеет интересные свойства. Например, отношение соседних чисел в ней стремится к замечательной математической константе — золотому сечению.
Последовательность очень быстро возрастает. Поэтому для вычислений чисел Фибоначчи лучше пользоваться вместительным типом — std::uint64_t.
Что нужно сделать
Напишите функцию GetFibonacci, которая вычисляет n-е по счёту число Фибоначчи.
Ограничения
Функция должна принимать на вход параметр типа int.
Функция должна возвращать результат типа std::uint64_t — число Фибоначчи с указанным номером.
Как будет тестироваться код
Тесты проверят, корректно ли реализуется функция и выполнены ли ограничения. 
Функция main при проверке учитываться не будет.
