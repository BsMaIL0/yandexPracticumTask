Представьте, что вы набрали длинный текст, но не заметили, что у вас включён режим Caps Lock. Все заглавные буквы набрались строчными, а строчные — заглавными. Перенабирать текст вам совершенно не хочется: давайте лучше напишем функцию, которая может исправить такой текст. Это будет функция SwitchCase.
Например, выражение SwitchCase("hELLO, wORLD!") должно иметь значение Hello, World!. Выражение SwitchCase("c++ is the best") должно иметь значение C++ IS THE BEST.
Также можно вызывать SwitchCase и для отдельных символов: SwitchCase('a') == 'A', SwitchCase('T') == 't', SwitchCase('!') == '!'.
Что нужно сделать
Используйте перегрузку, чтобы написать два варианта функции SwitchCase.
Для работы с символами (char).
Для работы со строками (std::string).
Первая функция должна сделать следующее:
- Если передан символ в верхнем регистре, то вызвать для него std::tolower и возвратить.
- Если передан символ в нижнем регистре, то вызвать для него std::toupper и возвратить.
- Иначе возвратить исходный символ.
Для проверки того, что символ в верхнем регистре, используйте функцию bool CheckIsUpper(char c), а функцию bool CheckIsLower(char c) — для проверки того, что символ в нижнем регистре.
Функция SwitchCase для работы со строками должна пройтись циклом по всем символам строки и преобразовать их регистр, используя первую версию SwitchCase.
Ограничения
В вашей программе должно быть две функции SwitchCase.
Первая функция SwitchCase принимает один параметр типа char и возвращает результат типа char.
Вторая функция SwitchCase принимает один параметр типа std::string и возвращает результат типа std::string.
Вторая функция SwitchCase принимает std::string по значению, а не по константной ссылке. Благодаря этому она может менять свой аргумент.
В функции SwitchCase для работы со строками используйте функцию SwitchCase для работы с символами.
