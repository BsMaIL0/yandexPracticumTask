Перегрузка позволяет не только по-разному обрабатывать значения разных типов, но и использовать тип для выбора нужной функции.
Что нужно сделать
Требуется создать функцию std::string TypeToString(<какой-то тип> arg), которая преобразует тип выражения в строку. Она должна поддерживать следующие типы:
int
char
double
float
std::string
И векторы от этих типов:
std::vector<int>
std::vector<char>
std::vector<double>
std::vector<float>
std::vector<std::string>
У вас должно получиться десять функций, например такая:

std::string TypeToString([[maybe_unused]] int arg) {
    return "int";
} 
Она уже дана в заготовке. 
Используйте перегрузку, чтобы реализовать остальные варианты.
Конструкция [[maybe_unused]] нужна, чтобы избежать предупреждения о неиспользуемом аргументе. 
Вместо неё можно просто не писать имя аргумента: std::string TypeToString(int), но такой способ считается менее современным.
