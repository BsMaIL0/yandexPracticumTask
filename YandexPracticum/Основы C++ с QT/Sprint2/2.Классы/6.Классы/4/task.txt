Не забудьте выключить телевизор
В этом задании вам нужно разработать класс по техническому заданию.
Что нужно сделать
Разработайте класс TV, моделирующий работу телевизора по следующим правилам:
Телевизор может находиться в одном из двух состояний: включённом или выключенном. Начальное состояние телевизора — выключен.
Когда телевизор включён, он позволяет выбрать канал с 1-го по 99-й включительно. Нельзя выбрать канал с номером меньше 1 и больше 99. Выключенный телевизор не может переключать каналы.
При первом включении в телевизоре выбран канал № 1. При последующем включении телевизор показывает тот канал, который был выбран перед его выключением.
В заготовке кода вам дан каркас класса TV. В классе объявлены методы:
TurnOn — включает телевизор.
TurnOff — выключает телевизор.
IsTurnedOn — возвращает информацию о том, включён телевизор (true) или нет (false).
SelectChannel — выбирает канал с заданным номером. Возвращает true в случае успеха и false при неудаче.
GetCurrentChannel — возвращает текущий номер канала. Если телевизор в данный момент выключен, возвращает 0.
Напишите реализацию этих методов, не меняя их сигнатуры. При реализации класса вам понадобится добавить поля. Объявите их в приватной области.
Как будет тестироваться код
В этой задаче для самопроверки представлен новый способ тестирования — юнит-тесты. Это значит, что вам не придётся многократно запускать код, чтобы подставить тестовые данные: все тесты сразу включены в функцию main. Если какой-то из тестов не пройдёт, вы увидите сообщение с указанием номера строки.
Например, если вы запустите код из заготовки, то получите такое сообщение:

submission: /exposed/submission/main.cpp:52: int main(): Assertion `tv.IsTurnedOn()' failed 
В этом сообщении указано, что ошибка произошла в функции main, на 52-й строке. Содержимое макроса assert — tv.IsTurnedOn() вернуло false. Если такое случается, программа останавливается. Остальные проверки не будут запущены. Макрос assert определён в заголовочном файле cassert.
Вы можете добавить свои тесты похожим образом. При проверке кода тренажёром тело функции main будет проигнорировано.
Юнит-тесты
Тест 1
Проверяет, что телевизор выключен по умолчанию.

TV tv;
assert(!tv.IsTurnedOn()); 
Тест 2
Проверяет работу методов TurnOn и TurnOff.

TV tv;
assert(!tv.IsTurnedOn());
tv.TurnOn();
assert(tv.IsTurnedOn());
tv.TurnOff();
assert(!tv.IsTurnedOn()); 
Тест 3
Проверяет работу метода GetCurrentChannel.

TV tv;
assert(!tv.IsTurnedOn());
assert(tv.GetCurrentChannel() == 0);
tv.TurnOn();
assert(tv.GetCurrentChannel() == 1);
tv.TurnOff();
assert(tv.GetCurrentChannel() == 0); 
Тест 4
Проверяет включение разных каналов.

TV tv;
tv.TurnOn();
for (auto i = 0; i < 10; ++i) {
    int channel = (71 * i) % MAX_CHANNEL + 1;
    assert(tv.SelectChannel(channel));

} 
Тест 5
Проверяет, что канал не будет включён, если телевизор выключен.

TV tv;
tv.TurnOff();
int channel = 50;
assert(!tv.SelectChannel(channel));
assert(tv.GetCurrentChannel() == 0); 
Тест 6
Проверяет включение разных каналов и их сохранение при выключении телевизора.

TV tv;
tv.TurnOn();
for (auto i = 0; i < 10; ++i) {
    int channel = (641 * i) % MAX_CHANNEL + 1;
    assert(tv.SelectChannel(channel));
    assert(tv.GetCurrentChannel() == channel);
    tv.TurnOff();
    assert(!tv.IsTurnedOn());
    assert(tv.GetCurrentChannel() == 0);
    tv.TurnOn();
    assert(tv.GetCurrentChannel() == channel);
}
