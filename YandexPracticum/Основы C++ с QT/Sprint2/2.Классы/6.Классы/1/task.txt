Летят утки
Утки Тихокряк (Whisper Quack) и Огнекрыл (Fire Wing) были частью утиного отряда, отправленного в дальний полёт. Утиный командир по рации отдаёт им приказы, сколько каждая из уток должна пролететь. Каждая утка, выполнив приказ, отчитывается об общем расстоянии, которое она пролетела. 
В заготовке кода вам дан класс Duck, моделирующий поведение утки. В классе два метода:
void SetName(const std::string& name) — задаёт имя утки. Этот метод написан полностью.
void Fly(int distance) — отдаёт утке команду пролететь указанную дистанцию. Дистанция задаётся в километрах. Этот метод нужно доработать.
Что нужно сделать
Доработайте метод Fly, чтобы при его вызове в std::cout выводилась строка:
{D} flies {M}km. Total flight distance is {T}km. 
Где {D} — имя утки, у которой вызван метод, {M} — расстояние, которое было приказано пролететь, {T} — общее расстояние, которое пролетела эта утка за всё время.
Расстояние, преодолённое уткой, должно храниться в приватном поле объекта утки.
Как будет тестироваться код
Тренажёр проверит, что программа работает правильно. При тестировании будет использовать main, данный в заготовке. Он использует формат данных, описанный ниже.
Формат ввода-вывода
Функция main из заготовки читает следующие данные:
Число int — количество команд.
Сами команды.
Каждая команда состоит из двух чисел:
Число int — номер утки (1 или 2).
Число int — расстояние, которое нужно пролететь утке.
Для каждой команды вызывается метод Fly нужной утки.
