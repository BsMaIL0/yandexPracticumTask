Летят утки 2
К Тихокряку и Огнекрылу присоединились другие утки. Теперь их целая семья. Точное количество уток и их имена программа узнает из стандартного ввода.
Что нужно сделать
Не меняя класс Duck, доработайте main, чтобы он мог обработать произвольное количество уток. Для удобства сложите уток в переменную типа std::vector<Duck>.
Как будет тестироваться код
Тренажёр проверит, что программа работает в соответствии с описанием и форматом, описанным ниже.
Формат ввода-вывода
Программа читает следующие данные:
Число int — количество уток (N).
Имена уток (по одному имени на строке).
Число int — количество команд.
Сами команды.
Каждая команда состоит из двух чисел:
Число int — номер утки. В отличие от предыдущей задачи, может быть любым числом в пределах количества уток (от 0 и до N - 1).
Число int — расстояние, которое нужно пролететь утке.
Для каждой команды вызывается метод Fly нужной утки.
