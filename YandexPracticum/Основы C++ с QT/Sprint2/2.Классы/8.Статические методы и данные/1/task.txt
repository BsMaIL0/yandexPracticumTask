Автомобиль
В заготовке вы найдёте класс, эмулирующий работу автомобиля с автоматической коробкой передач. Она сама переключает передачи в зависимости от скорости движения, поэтому водитель просто включает один из трёх режимов: «ехать вперёд» (режим DRIVE), «ехать назад» (режим REVERSE) и «парковка» (режим PARKED).
Двигатель автомобиля можно включить или выключить. Изначально двигатель автомобиля выключен.
Включив режим движения, можно менять скорость автомобиля. Изначально скорость равна нулю.
Передача автомобиля задаётся enum-типом Gear. В нём перечислены все возможные режимы.
Изначальная передача Gear::PARKED. Только на этой передаче можно выключать двигатель.
Если двигатель выключен, то изменить ни скорость, ни передачу нельзя.
Автомобиль имеет следующий набор методов:
void TurnEngineOn(). Включает двигатель автомобиля.
bool TurnEngineOff(). Выключает двигатель автомобиля. Двигатель можно выключить только на передаче Gear::PARKED.
bool IsEngineOn(). Проверяет, включён ли двигатель. Возвращает true, если двигатель включён, и false — в противном случае.
Gear GetGear(). Возвращает активную передачу.
bool SetGear(Gear gear). Устанавливает передачу. Её можно изменить только при включённом двигателе, когда автомобиль стоит на месте.
int GetSpeed(). Возвращает текущую скорость движения автомобиля.
Direction GetDirection(). Возвращает направление движения автомобиля: стоит на месте, едет вперёд или назад. Автомобиль стоит на месте, если он находится на передаче Gear::PARKED или его скорость равна нулю. В ином случае направление зависит от передач Gear::DRIVE и Gear::REVERSE. Метод должен возвращать значение перечислимого типа Direction.
bool SetSpeed(int speed). Устанавливает скорость. Скорость можно изменить, только если желаемая скорость совместима с текущей передачей. Например, в режиме парковки нельзя установить скорость, отличную от нуля, а при движении назад или вперёд нельзя ехать быстрее максимально допустимой скорости.
Статический метод int GetMaxSpeedForGear(Gear gear). Возвращает максимально допустимую скорость на указанной передаче.
Статический метод bool IsSpeedValidForGear(int speed, Gear gear). Проверяет, совместима ли скорость с передачей. Скорость для передач должна быть в пределах от 0 до значения, возвращённого методом GetMaxSpeedForGear.
Метод GetMaxSpeedForGear должен возвращать следующие значения:
Car::MAX_SPEED для режима DRIVE;
Car::MAX_REVERSE_SPEED для режима REVERSE;
0 для режима PARKED.
Ряд методов возвращают флаг успеха. То есть true, если операция удалась, и false — в противном случае. Это следующие методы: SetGear, SetSpeed, TurnEngineOff.
Некоторые методы даны в заготовке, некоторые нужно реализовать для прохождения задачи.
Что нужно сделать
Добавьте реализацию недостающих методов автомобиля:
TurnEngineOn;
TurnEngineOff;
GetDirection;
статический метод IsSpeedValidForGear;
статический метод GetMaxSpeedForGear.
Работайте только в файле car.cpp. Учитывайте, что слово static пишется в .h-файле при объявлении, но не пишется в .cpp-файле при определении.
Ограничения
Метод IsSpeedValidForGear должен использовать GetMaxSpeedForGear, чтобы проверить корректность скорости.
Разместите реализацию методов в .cpp-файле.
Как будет тестироваться код
Задача будет проверена юнит-тестами, аналогичными тем, что находятся в функции main в заготовке.
