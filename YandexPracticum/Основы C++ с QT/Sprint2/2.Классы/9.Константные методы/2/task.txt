В заготовке дан класс Time, который хранит момент времени в течение суток с точностью до секунды. Вам нужно дописать недостающие методы этого класса.
Класс использует переменную time_offset_, чтобы хранить время. Она показывает количество секунд с начала дня. С помощью time_offset_ можно вычислить компоненты времени — часы, минуты и секунды:
Если time_offset_ == 0, то время 00:00:00.
Если time_offset_ == 42, то время 00:00:42.
Если time_offset_ == 65, то время 00:01:05.
Если time_offset_ == 3600, то время 01:00:00, так как в одном часе 3600 секунд.
Вычислить компоненты времени можно так:
Чтобы вычислить секунды, нужно взять остаток time_offset_ от деления на 60.
Чтобы вычислить минуты, нужно поделить time_offset_ на 60 и взять остаток результата от деления на 60.
Чтобы вычислить часы, нужно поделить time_offset_ на 3600.
Эти формулы будут работать, так как в C++ при делении целых чисел идёт округление в сторону нуля. Напомним, чтобы взять остаток от деления, нужно использовать операцию %.
Для обратного преобразования (часов, минут и секунд в секунды с начала суток) нужно применить такую формулу: часы * 3600 + минуты * 60 + секунды.
Что нужно сделать
Допишите недостающие методы этого класса:
Метод void SetTime(int hours, int minutes, int seconds). Изменяет текущее время, переводя время, заданное тремя компонентами, в секунды, и сохраняя их в time_offset_.
Методы int GetHours(), и int GetSeconds(). Возвращают количество часов и секунд в текущем объекте: значение часов от 0 до 23, значение минут и секунд в диапазоне от 0 до 59.
Метод int GetDistance(Time from, Time to). Возвращает расстояние (в секундах) от момента from до момента to. Чтобы его вычислить, нужно из количества секунд в объекте to вычесть количество секунд в объекте from.
В заготовке даны методы int GetMinutes() и void AddSeconds(int offset).
Для каждого метода класса выберите правильный тип: обычный, константный или статический. При этом вы можете ориентироваться на вопрос об этом классе из урока.
При вызове SetTime количество часов находится в пределах от 0 до 23 включительно, количество минут и секунд от 0 до 59. Об этом заботится пользователь метода.
