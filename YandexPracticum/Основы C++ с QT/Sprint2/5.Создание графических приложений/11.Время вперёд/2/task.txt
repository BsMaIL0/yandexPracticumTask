Откройте проект из папки sprint2/alarm вашего репозитория.
Счастливые люди обходятся без часов. Разнообразные программы-напоминалки явно должны прибавлять счастья, ведь позволяют не следить за временем! В этом уроке реализуем такое приложение, создадим «Будильник», который с помощью выбранной мелодии должен сделать начало нового дня прекраснее.
В уроках этой темы вы изучили QTimer, QMenu, QFileDialog и освоили проигрывание аудиофайлов, что пригодится вам при решении.
Программа «Будильник»:
Показывает текущее время.
Позволяет установить будильник на любое заданное время.
Отсчитывает время до будильника.
Когда приходит время, проигрывает выбранную мелодию.
Кнопка «Стоп» останавливает время или мелодию, если мелодия уже запущена.
Что нужно сделать
Отображение текущего времени
Чтобы обновлять текущее время, вам понадобится prac::QTimer. Добавьте его в класс окна.
Запускать таймер можно в слоте OnTimer(). Добавьте этот слот в MainWindow. В конструкторе соедините его с сигналом таймера и вызовите, чтобы выполнить первое срабатывание. В слоте сделайте следущее:
Определите текущее время, воспользовавшись prac::QTime::currentTime().
Преобразуйте время в строку, используя метод TimeToString.
Поместите эту строку в метку lbl_now.
Запустите таймер до следующего срабатывания.
Чтобы вычислить время до следующего срабатывания, вычтите из 1000 количество миллисекунд текущего времени: 1000 - now.msec(). Таким образом, вы запустите таймер до начала следующей секунды.
После выполнения этих действий программа начнёт показывать текущее время.
Установка будильника
При нажатии кнопки pb_start_stop нужно запустить будильник. При этом нужно вычислить момент срабатывания будильника. Чтобы его хранить, используйте переменную alarm_moment_ типа QDateTime.
Значение этой переменной будут проверять тесты с помощью метода GetAlarmTime();
Время срабатывания будильника можно вычислить так:
Получить текущие время и дату с помощью auto now = prac::QDateTime::currentDateTime().
Получить время будильника (только время, без даты) так: QTime(ui->sb_hour->value(), ui->sb_min->value(), ui->sb_sec->value()).
Соединить время будильника и текущую дату: alarm_moment_ = QDateTime(now.date(), alarm_time).
Если время будильника находится в прошлом (то есть меньше now), то прибавить один день: alarm_moment_ = alarm_moment_.addDays(1).
Также смените надпись на кнопке на слово "Стоп".
После выполнения этих действий программа будет обрабатывать нажатие кнопки "Старт".
Отображение времени до будильника
Добавьте в класс метод ProcessAlarm, который будет отображать время до будильника. Вызывайте этот метод из слота таймера OnTimer и из слота нажатия кнопки.
Убедиться, что будильник установлен, можно сравнив время будильника со значением по умолчанию:

if (alarm_moment_ != QDateTime{}) {
    // Будильник установлен.
} else {
    // Будильник не установлен.
} 
Если будильник не установлен, выведите в lbl_timeout текст "Установите будильник".
Если будильник установлен:
Получите текущий момент времени.
Вычтите его из времени будильника: auto time_to_alarm = alarm_moment_ - now. Вы получите временной интервал.
Преобразуйте его в строку, используя TimeToString и выведите в lbl_timeout.
После выполнения этих действий программа начнёт показывать время до будильника.
Остановка будильника
В слот нажатия кнопки pb_start_stop добавьте проверку того, что будильник не установлен. Если будильник установлен, сбросьте время будильника в значение по умолчанию: alarm_moment_ = QDateTime{} и верните надпись "Старт" на кнопку.
Вызовите метод ProcessAlarm, чтобы установить правильный текст в lbl_timeout.
Выбор мелодии
Создайте слот для обработки нажатия меню action_load_file. В этом слоте попросите пользователя выбрать файл таким образом:

QString file_name = prac::QFileDialog::getOpenFileName(this,
                            QString::fromUtf8("Открыть файл"),
                            QDir::currentPath(),
                            "*.wav;*.mp3"); 
Вот пример мелодии, которую вы можете использовать для теста:
alarm.mp3
Добавьте в MainWindow переменные типов prac::QMediaPlayer и QAudioOutput. В конструкторе привяжите аудиовыход к плееру.
При выборе файла сохраните его в QMediaPlayer методом setSource и выведите имя файла в lbl_melody.
Срабатывание будильника
В ProcessAlarm добавьте проверку того, что наступило время будильника. Для этого сравните текущий момент времени и alarm_moment_. Это нужно делать только, если задано alarm_moment_.
Если alarm_moment_ находится в прошлом, то нужно заменить надпись в lbl_timeout на текст "Пора вставать". Запустите мелодию, если плеер её не воспроизводит:

if (player_.playbackState() != QMediaPlayer::PlaybackState::PlayingState) {
    player_.play();
} 
После выполнения этих действий будильник начнёт воспроизводить выбранную мелодию при наступлении нужного времени.
