Это первая задача из серии о фотоальбоме. Здесь нужно показать лишь одну фотографию, которая должна загружаться из ресурсов.
В качестве ресурса возьмите картинку «Котик в ночи», созданную в приложении Шедеврум.

Сейчас будем подгонять не размер формы под размер изображения, а, наоборот, размер изображения под размер области на форме. Причём картинка будет адаптироваться к изменениям с помощью обработки события onResize.
В заготовке будет подготовленная форма:
у неё удалена строка состояния;
настроена палитра (чёрный цвет фона и белый цвет надписей);
добавлена надпись lbl_img для отображения картинки.
В этой задаче важно правильно масштабировать и центрировать изображение.
В классе окна создано поле QPixmap active_pixmap, которое хранит изображение, отображаемое на форме. При изменении размеров окна это изображение должно скалироваться, затем отображаться на форме.
Для изменения размеров отображаемого изображения у класса QPixmap есть метод scaledToWidth, который масштабирует изображение до нужной ширины, и scaledToHeight, который масштабирует изображение до нужной высоты. Методы не меняют изображение, а возвращают изменённую копию.
Вам нужно будет реализовать метод FitImage. Он должен скалировать изображение и отобразить его в нужном месте на форме. Метод будет использовать изображение из поля active_pixmap. Он должен вызывать функцию ResizeImgToFit, чтобы получить изображение нужного размера. Затем это изображение нужно поместить в lbl_img и центрировать.
Чтобы изображение оказалось по центру, нужно вычислить координаты виджета и переместить его методом move. Вычислить координаты можно этими формулами:

// win_w, win_h — размеры окна.
// img_w, img_h — размеры скалированного изображения.
int lbl_x = (win_w - img_w) / 2; // Координата x.
int lbl_y = (win_h - img_h) / 2; // Координата y. 
Также нужно изменить размеры виджета, чтобы они совпали с размерами скалированного изображения.

Метод FitImage должен выполнять следующее:
Вызывать функцию ResizeImgToFit, передавая в качестве параметра active_pixmap. Результат функции – скалированное изображение. Его нужно поместить в локальную переменную.
Помещать скалированное изображение в lbl_img.
Изменять размер lbl_img до размера скалированного изображения методом resize (ui->lbl_img->resize).
Методом move перемещать lbl_img для центрирования изображения. На этом шаге воспользуйтесь формулами, данными выше.
