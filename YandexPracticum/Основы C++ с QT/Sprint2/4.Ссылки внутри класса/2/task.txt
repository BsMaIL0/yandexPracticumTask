Снова пульт
В предыдущей теме вы уже создавали класс TVControl. Он представлял собой набор функций для управления любым телевизором. В этом задании вам предстоит разработать класс пульта, управляющий только одним телевизором, ссылка на который будет передаваться в конструктор класса.
Что нужно сделать
Добавьте внутрь TVControl поле-ссылку на класс TV, чтобы связать пульт управления с телевизором.
Чтобы проинициализировать ссылку, добавьте в TVControl конструктор, принимающий ссылку на телевизор.
Добавьте и реализуйте приватные методы класса для обработки соответствующих команд. Описание команд представлено ниже. Оно не изменилось.
По-прежнему важно не вызывать методы телевизора в ReadAndApplyCommand. Вместо этого следует использовать соответствующие одноимённые методы класса TVControl для выполнения команд телевизора.
Взаимодействие с телевизором должно осуществляться так:

int main() {
    TV tv;
    const TVControl control{tv};
    while (control.ReadAndApplyCommand()) {
    }
} 
Вы можете использовать решение прошлой задачи про пульт. Но будьте внимательны: методы должны стать нестатическими и больше не должны принимать ссылку на телевизор.
Описание команд
Программа должна поддерживать следующие команды:
Info. Выводит в std::cout информацию о телевизоре. Если телевизор включён, то выводит строку «TV is turned on. Channel is N», где N — номер канала. Если телевизор выключен, то выводит строку «TV is turned off». Обработка этой команды уже реализована в заготовке.
TurnOn. Включает телевизор и выводит строку «Turning on».
TurnOff. Выключает телевизор и выводит строку «TV is turned off».
SelectChannel {N}, где {N} — номер канала. N должно быть числом. Функция предназначена для выбора указанного канала на телевизоре. В случае успешного выбора выводит строку «Channel selected». Если канал не удалось выбрать из-за того, что телевизор был выключен, выводит строку «TV is not turned on». Если канал не удалось выбрать из-за того, что номер канала выходил за допустимые пределы, то выводит строку «Invalid channel».
Exit. Завершает диалог с пользователем и выходит из программы. Обработка этой команды реализована в заготовке.
Ограничения
Не меняйте класс TV.
Инициализируйте поле-ссылку на телевизор в конструкторе класса TVControl.
