Откройте проект из папки sprint2/hello-button вашего репозитория.


Скриншот приложения, которое нужно сделать
Что нужно сделать
На форму главного окна добавьте:
Поле для ввода QLineEdit. Назовите виджет le_input.
Метку QLabel для вывода результата. Назовите виджет lbl_result.
Кнопку.
По нажатию на кнопку в метке lbl_result должна появиться строка Привет из Qt, <имя>!, где вместо <имя> подставится текст, введённый в QLineEdit.
Чтобы этого добиться, напишите код в слоте нажатия на кнопку. Там нужно будет получить текст из le_input таким способом:

QString name = ui->le_input->text(); 
Затем нужно будет получить приветствие и поместить его в lbl_result:

ui->lbl_result->setText(greeting); 
Ограничения
На форме должно быть ровно одно поле QLineEdit.
На форме должна быть ровно одна кнопка QPushButton.
На форме должна быть метка QLabel с именем lbl_result.
Как будет тестироваться код
Будут запущены тесты Qt.
