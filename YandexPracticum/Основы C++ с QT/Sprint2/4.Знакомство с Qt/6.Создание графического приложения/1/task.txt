Что нужно сделать
Создайте приложение в Qt Creator c графическим окном, которое содержит две надписи:
«Hello» — синего цвета, курсивом, размер шрифта 30, выравнивание по левому краю;
«Qt» — красного цвета, жирным, размер шрифта 35, выравнивание по правому краю.
Назовите основной класс MyWindow, а файлы — mywindow.h, mywindow.ui, mywindow.cpp.
Скриншот

Ограничения
В вашем приложении должно быть только две надписи.
Отформатируйте текст с помощью HTML-тегов без использования CSS.
Что отправлять на проверку
Измените только файл mywindow.ui. Остальные оставьте без изменений.
Возможные проблемы
При изменении настроек через форматирование текста может возникнуть проблема, что через панель не будут меняться настройки текста.
