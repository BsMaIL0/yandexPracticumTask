(Не)обыкновенная дробь
Обыкновенная дробь представляет число в виде отношения двух целых чисел: числителя и знаменателя. Например, 3/8 (три восьмых), -7/25 (минус семь двадцать пятых). Знаменатель не может быть равен нулю.
Для удобства работы с дробями их приводят к несократимому виду. Дробь несократима, если её числитель и знаменатель не имеют общих делителей, помимо единицы. Например, дробь

Что нужно сделать
Создайте класс Rational, который представляет обыкновенную дробь, и две функции для сложения и вычитания дробей.
У Rational реализуйте конструкторы, создающие дробь тремя способами:
  
Без параметров. Создаёт дробь, равную нулю. Числитель такой дроби равен нулю, а знаменатель — единице.
С одним целочисленным параметром. Создаёт дробь, равную этому числу. Числитель такой дроби равен переданному параметру, а знаменатель — единице. Например, 5/1.
С двумя целочисленными параметрами — числителем и знаменателем. Создаёт несократимую дробь с положительным знаменателем. Если был передан нулевой знаменатель, вызовите std::abort, чтобы завершить работу программы.
Также реализуйте методы класса Rational:
int GetNumerator() const. Возвращает числитель дроби.
int GetDenominator() const. Возвращает знаменатель дроби.
double AsDouble() const. Возвращает представление дроби в виде числа с плавающей точкой.
Реализуйте отдельные функции для работы с дробями:
Rational Add(Rational r1, Rational r2). Возвращает несократимую дробь с положительным знаменателем, равную сумме дробей r1 и r2.
Rational Sub(Rational r1, Rational r2). Возвращает несократимую дробь с положительным знаменателем, равную разности дробей r1 и r2.

Юнит тесты
Для удобства тестирования в main реализована функция ToString(), которая возвращает строку типа {n}/{d}, где n — числитель, а d — знаменатель.
После завершения тестирования программа ожидает ввода четырёх чисел. Из этих чисел она создаёт две дроби и выполняет с ними арифметические операции: сложение и вычитание.
Тест
Код:

Rational r;
assert(ToString(r) == "0/1"s);
assert(r.AsDouble() == 0.0);

Конструктор по умолчанию создаёт дробь 0/1.

Как будет тестироваться ваш код
Тренажёр проверит работу класса Rational и функций Add, Sub тестами, аналогичными тем, что представлены в примерах.
Конструктор класса Rational не будет вызываться с нулевым знаменателем. Вызов std::abort не будет тестироваться.
