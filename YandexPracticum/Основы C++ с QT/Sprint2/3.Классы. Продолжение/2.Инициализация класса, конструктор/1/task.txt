Конструктор точек и треугольников
В заготовке кода даны структуры Point и Triangle. У них нет конструкторов, поэтому пользователи могут создавать точки и треугольники, указывая неполный набор параметров:

void PrintPoint(Point p) {
    ...
}

int main() {
    // Точка с координатами (42.0, 0.0).
    Point p{42.0};

    // Напечатает точку (42.0, 0).
    PrintPoint({42});

    // Треугольник с вершинами (10.5, 0.0), (3.5, 2.5), (0.0, 0.0).
    Triangle t{{10.5}, {3.5, 2.5}};
}
 
Такая свобода позволяет программисту писать код, который выглядит странно и вызывает вопросы при чтении.
Добавьте в структуры Point и Triangle конструкторы, ограничивающие способы создания точек и треугольников:
Точка может быть сконструирована указанием нуля или двух координат. Если координаты не указаны, точка должна иметь координаты (0, 0).
Треугольник может быть сконструирован указанием нуля или трёх вершин. Если вершины не переданы, создаётся вырожденный треугольник, у которого все вершины находятся в точке (0, 0).

// Разрешённые способы конструирования точек и треугольников.

Point p;           // Точка с координатами (0, 0).
Point q{3.5, 2.8}; // Точка с координатами (3.5, 2.8).

// Треугольник с вершинами (0, 0), (0, 0), (0, 0).
Triangle t;

// Треугольник с вершинами (3.5, 2.8), (0, 0), (-3.5, -2.8).
Triangle t2{ q, p, {-3.5, -2.8}};

Point p1;
assert(p1.x == 0);

Тест 2

Point p2{10, 20};
assert(p2.x == 10);
assert(p2.y == 20);

assert(t1.p1.x == 0);
assert(t1.p1.y == 0);

assert(t1.p2.y == 0);
assert(t1.p3.x == 0);
assert(t1.p3.y == 0);

assert(t2.p1.y == 20);
assert(t2.p2.x == 30);
assert(t2.p2.y == 40);
assert(t2.p3.x == 50);

Как будет тестироваться ваш код
Тренажёр проверит, что треугольник и точка могут быть созданы только перечисленными способами. Функция main проверяться не будет.
Возможные проблемы
У точки нет конструктора по умолчанию.
У точки нет конструктора, принимающего два параметра типа double.
У точки есть лишние конструкторы.
В конструкторе точки параметры не должны иметь значения по умолчанию.
Конструктор точки не инициализируется нулём.
Параметризованный конструктор точки неверно инициализирует её координаты.
У треугольника нет конструктора по умолчанию.
У треугольника нет конструктора, принимающего три точки.
У треугольника есть лишние конструкторы.
В конструкторе треугольника параметры не должны иметь значения по умолчанию.
В треугольнике конструктор по умолчанию не инициализирует вершины координатами (0, 0).
В треугольнике параметризованный конструктор неверно инициализирует вершины.
