Что нужно сделать
Напишите функцию CountAndPrint, которая считает, сколько раз в строке встречается заданный символ. 
В функции должно быть два параметра: строка (тип std::string) и искомый символ (тип char).
В качестве ответа нужно вывести сообщение В строке "<исходная строка>" символ '<переданный символ>' встречается <число> раз(а).. Считайте символ в разных регистрах двумя разными символами.
В поиске символа вам поможет метод find, который вы изучали в уроке по работе со строками. Напомним, что этот метод возвращает номер позиции для первого вхождения указанного символа или значение std::string::npos, если его в строке нет.

std::string str = "hello";
int pos_1 = str.find('l'); // pos_1 == 2. Первый символ 'l' стоит на позиции 2.
int pos_2 = str.find('*'); // pos_2 == std::string::npos. Символа '*' нет. 
Если нужно искать символ не во всей строке, а с какого-то места, функции find можно передать номер позиции, с которой следует начать поиск.

// Позиции:        01234567 
std::string str = "hello world!";

// Ищем первое 'o':
int pos_1 = str.find('o'); // pos_1 = 4. 

// Чтобы найти следующее, ищем с позиции 5:
int pos_2 = str.find('o', pos_1 + 1); // pos_1 = 7.  
Для решения задачи используйте переменную-счётчик и цикл. На каждой итерации нужно вызывать метод find, который будет искать новое вхождение символа, начиная с позиции, следующей за последним найденным символом. 
Например, если предыдущий символ находился на позиции pos, на новой итерации find должен начинать поиск с pos + 1.
Позицию, после которой нужно начать поиск в следующей итерации, нужно сохранить в переменную типа size_t. Вы можете выполнить первый поиск до цикла и сохранить его результат в эту переменную.
Такой подход позволит не учитывать символы, которые уже были посчитаны.
Цикл должен выполняться, пока в строке есть неподсчитанные искомые символы (т. е. позиция поиска не равна  std::string::npos).
Ограничения
Считайте символы, используя метод find.
