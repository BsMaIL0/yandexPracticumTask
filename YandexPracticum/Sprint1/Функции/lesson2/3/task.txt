В функциях можно не только писать обычные команды, но и вызывать другие функции. Потренируйтесь это делать, создав функцию для разбиения на подстроки с печатью информации о каждой подстроке.
Что нужно сделать
Напишите функцию SplitAndAnalyze, которая:
Получает строку.
Делит её на подстроки по заданному символу.
Для каждой из подстрок вызывает CountAndPrint.
Например, для исходной строки hello_beautiful_world и символа _ после разделения получатся подстроки hello, beautiful и world. Обратите внимание, что разделитель не входит ни в одну из них. 
У SplitAndAnalyze будет три параметра:
Строка для разбиения (std::string).
Символ, по которому будет делиться строка (char).
Символ, который будет считаться в подстроках (char).
Если в исходной строке символ-разделитель не встречается ни разу, считайте, что она состоит из одной подстроки — самой себя.
Для определения границ подстроки используйте тот же принцип, что и в предыдущей задаче: создайте цикл с методом find. Однако теперь вместо увеличения счётчика нужно вызывать функцию CountAndPrint, передавая ей текущую подстроку.
Получить подстроку поможет метод substr, который вырезает из строки часть, начинающуюся с указанной позиции и содержащую указанное количество символов. Например:

std::string str = "hello world!";
std::string small_str = str.substr(3, 6); // small_str = "lo wor".

// small_str содержит шесть символов из str, начиная с третьего. 
Если использовать substr без второго параметра, она вернёт символы с указанной позиции и до конца строки.

std::string str = "hello world!";
std::string small_str = str.substr(3); // small_str = "lo world!". 
Алгоритм разделения на подстроки будет следующим:
Объявите переменную start для хранения позиции начала подстроки. В начале её значение равно 0.
Объявите переменную end для хранения позиции конца подстроки. Она равна позиции первого вхождения символа-разделителя.
Используйте переменные start и end, чтобы вырезать из строки подстроку с помощью substr. Чтобы узнать длину подстроки, нужно вычесть позицию начала из позиции конца.
Выведите информацию о подстроке согласно условию.
Обновите переменную start — теперь она должна указывать на символ, идущий после обработанной строки, т. е. равняться end + 1.
Обновите переменную end — она должна указывать на позицию очередного символа, найденного с помощью find. Начинать поиск нужно с позиции start.
Повторяйте действия 3–6, пока не дойдёте до конца строки. В этом случае find не найдёт новых подстрок и вернёт std::string::npos. Проверку условия можно реализовать через цикл while.
Цикл завершился, когда последний find возвратил npos. В этом случае нужно обработать оставшуюся часть строки, начиная с позиции start. Для её получения используется метод substr с одним параметром.
 Код для этого можно расположить после цикла. Он также обработает ситуацию, когда нужный символ вообще не найден в строке и цикл не совершил ни одной итерации. Например:

char delimiter = '_';
std::string str = "hello_world_";
