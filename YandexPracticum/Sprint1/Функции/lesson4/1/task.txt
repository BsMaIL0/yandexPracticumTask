Переводить числа из одной системы счисления в другую — не такая простая задача. Начнём с отдельных цифр. Напомним, что цифрой может быть и буква, если основание системы счисления больше десяти. Вот, например, значения некоторых цифр:
0 — 0,
9 — 9,
A — 10,
b — 11,
z — 35.
Что нужно сделать
Напишите функцию int CharToDigit(char ch), которая рассматривает символ ch как цифру и возвращает её значение в виде числа типа int. Символ ch может быть десятичной цифрой или латинской буквой. Функция должна работать как со строчными, так и с заглавными буквами. Например, численные значения букв 'A' и 'a' равны 10.
При реализации функции требуется рассмотреть три случая: 
ch — десятичная цифра.
ch — заглавная буква.
ch — строчная буква.
Если символ ch — десятичная цифра, например '5', её математическое значение можно получить с помощью таблицы ASCII. Для этого достаточно вычесть из кода ch код символа '0': int val = ch - '0'.
Если же символ ch является заглавной буквой, алгоритм преобразования будет следующим:
Вычесть из ch код символа A.
Прибавить к результату 10.
Аналогично со строчными буквами, но вместо 'A' нужно вычитать 'a'.
При вычитании символов результат автоматически преобразуется в число. Например:

// Численные значения букв: a — 10, b — 11, c — 12, d — 13, e — 14.
char c = 'e';

int pos = c - 'a'; // pos == 4.
int val = pos + 10; // val = 14, верное значение буквы 'e'. 
Заготовка функции main прочитает один символ из std::cin, преобразует его функцией CharToDigit в число и выведет в десятичной системе счисления.
Как будет тестироваться код
Тренажёр вызовет функцию CharToDigit с различными входными значениями и сравнит результат с ожидаемым.
Гарантируется, что символ для перевода находится в одном из диапазонов:
цифры от 0 до 9,
заглавные буквы от A до Z,
строчные буквы от a до z.
