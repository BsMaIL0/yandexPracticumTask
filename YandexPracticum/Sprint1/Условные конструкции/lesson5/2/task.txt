Что нужно сделать
Напишите программу, которая считает из потока ввода целое число N и определит, чётное оно или нет. Если число чётное, нужно вывести строку «Число N чётное», в противном случае — строку «Число N нечётное», заменив символ N прочитанным числом. 
Ограничения
В этой задаче нельзя использовать оператор if.
Как будет тестироваться код
Вывод вашей программы будет сравниваться с эталонным.
Примеры
Пример 1
Входные данные
5 
Результат
Число 5 нечётное 
Пример 2
Входные данные
2 
Результат
Число 2 чётное 
Пример 3
Входные данные
0 
Результат
Число 0 чётное 
Пример 4
Входные данные
-9 
Результат
Число -9 нечётное
