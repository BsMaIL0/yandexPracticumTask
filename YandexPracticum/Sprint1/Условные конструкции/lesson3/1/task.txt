В прошлом уроке вы находили максимум из двух чисел. Теперь задача усложняется — перед вами три числа.
Что нужно сделать
Напишите программу, которая считает из потока ввода три целых числа — N, M, K — и выведет в поток вывода наименьшее и наибольшее из этих чисел. Ответ нужно вывести в строку через пробел.
Как будет тестироваться код
На вход будут подаваться числа N, M, K — любые целые числа, умещающиеся в int.
Вывод вашей программы будет сравниваться с эталонным.
Примеры
Пример 1
Входные данные
1 4 0 
Результат
0 4 
Объяснение
Программа вывела 0 и 4, так как 0 — минимальное число (0 < 1 и 0 < 4), а 4 — максимальное число (4 > 1 и 4 > 0).
Пример 2
Входные данные
2 3 3 
Результат
2 3 
Объяснение
Программа вывела 2 и 3, так как 2 — минимальное число (2 < 3), а 3 — максимальное число (3 > 2).
Пример 3
Входные данные
50 -111 51 
Результат
-111 51 
Пример 4
Входные данные
0 -1 -1 
Результат
-1 0
