Вы уже видели, что C++ умеет считать. Но одно дело — наперёд заданные числа, другое — любые, которые введёт пользователь. Программа, которая получится в результате решения этой задачи, будет выполнять четыре арифметические операции с произвольными числами, полученными из ввода.
Что нужно сделать
Создайте две переменные типа double. Прочитайте их значения из std::cin. Выведите в std::cout сумму, разность, произведение и отношение прочитанных чисел. Всего выведите четыре строки:
Сумма: <результат сложения>
Разность: <результат вычитания>
Произведение: <результат умножения>
Отношение: <результат деления> 
Для суммы используйте операцию +, для разности — -, для произведения — *, для отношения — /.
Все команды записывайте внутри функции main.
Ограничения
Не выводите ничего лишнего в std::cout. Команда ввода должна идти до первой команды вывода.
Выводите результаты именно в том порядке, в каком требуется в условии.
Обратите внимание на пробелы и знаки пунктуации, они тоже имеют значение.
Как будет тестироваться код
На вход будут подаваться числа с плавающей точкой. Второе число никогда не равно нулю. Будет проверено, что программа всегда выдаёт верный текст и не пишет ничего лишнего.
Примеры
Пример 1
Входные данные
15 3 
Результат
Сумма: 18
Разность: 12
Произведение: 45
Отношение: 5 
Пример 2
Входные данные
0 -3.5 
Результат
Сумма: -3.5
Разность: 3.5
Произведение: -0
Отношение: -0 
Пример 3
Входные данные
0.5 2.2 
Результат
Сумма: 2.7
Разность: -1.7
Произведение: 1.1
Отношение: 0.227273
