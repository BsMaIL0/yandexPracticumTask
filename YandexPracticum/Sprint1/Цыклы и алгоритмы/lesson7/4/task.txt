Уникальные элементы
Это необязательное задание. Оно не блокирует прохождение курса.
Вы можете выполнить его сейчас или вернуться позже. Чтобы пропустить задание, нажмите кнопку «Проверить».
В вектор можно добавить два одинаковых элемента. Даже не два — не исключены большие векторы, целиком состоящие из одинаковых чисел. А вот если элемент в векторе встречается только один раз, то такой элемент называется уникальным.
Ваша задача — посчитать количество уникальных элементов вектора.
Что нужно сделать
Напишите программу, которая считает число N (N > 0) и ряд из N целых чисел. Программа должна вывести количество уникальных чисел в ряду.
У этой задачи есть различные решения. Не стремитесь получить самое эффективное сейчас, вы ещё узнаете интересные приёмы. Ищите один из возможных алгоритмов в подсказке.
Как будет тестироваться код
Вывод вашей программы будет сравниваться с эталонным.
Примеры
Пример 1
Входные данные
3
1 2 3 
Результат
3 
Объяснение
Программа вывела 3, так как в ряду три числа представлены по одному разу.
Пример 2
Входные данные
3
1 2 1 
Результат
1 
Объяснение
Одно уникальное число: 2.
Пример 3
Входные данные
1
100 
Результат
1 
Объяснение
Одно число, оно автоматически уникальное.
Пример 4
Входные данные
0
 
Результат
0 
Объяснение
Ни одного числа — ни одного уникального.
Пример 5
Входные данные
6
1 42 100500 1 42 100500 
Результат
0 
Объяснение
Каждое число повторяется дважды.
Пример 6
Входные данные
10
1 42 100500 12 1 42 100500 5 3 1 
Результат
3 
Объяснение
Три уникальных числа — 12, 5 и 3.
