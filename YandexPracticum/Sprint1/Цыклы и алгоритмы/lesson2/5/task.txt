Если у вашего англоговорящего друга день рождения, можно написать просто: happy birthday! (англ. «с днём рождения»). Но HAPPY BIRTHDAY! смотрится гораздо веселей!
Напишите программу, которая изменит регистр символов в строке.
Что нужно сделать
Считайте 2 строки из потока ввода. Первая строка — тип регистра, к которому нужно преобразовать слово: lower — нижний регистр, upper — верхний регистр. Вторая строка — ASCII-строка, в которой нужно изменить регистр символов. Программа должна преобразовать все символы строки к указанному регистру и вывести строку.
Алгоритм решения
В программе используйте цикл, чтобы пройти по всем символам строки:

// Переменная-счётчик типа size_t проходит по 
// индексам символов — от 0 до str.size() - 1.
size_t i = 0;
while (i < str.size()) {
    ...
    ++i;
} 
Внутри цикла вам нужно:
Получить символ строки с нужным номером.
Проверить, является ли он буквой.
Если да — преобразовать к верхнему или нижнему регистру.
Изменить символ в строке.
В конце не забудьте вывести изменённую строку.
Получить символ по номеру можно операцией []:

char letter = str[i]; 
Проверить, что символ — буква, можно функцией isalpha из библиотеки <cctype>:

// Если letter — буква.
if (std::isalpha(letter)) {
    ...
}

// Другой способ:
if ((letter >= 'A' && letter <= 'Z') ||
    (letter >= 'a' && letter <= 'z')) {
    ...
} 
Преобразовать к верхнему регистру можно функцией toupper, к нижнему — tolower.

// К верхнему регистру.
letter = toupper(letter);

// К нижнему регистру.
letter = tolower(letter); 
Для изменения символа в строке пригодится та же операция []:

str[i] = letter; 
Как будет тестироваться код
Вывод вашей программы будет сравниваться с эталонным.
