Одно из важных применений циклов — подсчёт, в том числе подсчёт чисел. Например, в строке В эпичной битве сразились 40 витязей прекрасных, 300 спартанцев и 40 разбойников всего три числа: 40, 300 и ещё раз 40. 
В этой задаче вам предстоит реализовать алгоритм для подсчёта чисел в строке. Для решения вам пригодится концепция стейт-машины с такими стейтами:
Обычный стейт.
Стейт «внутри числа».
Что нужно сделать
Напишите программу, которая прочитает строку функцией std::getline и посчитает количество чисел в ней. Числа могут состоять из любого количества подряд идущих цифр.

if (std::isdigit(line[i])) {
    ...
} else {
    ...
} 
Числом может быть любая последовательность цифр, идущих подряд. В этой задаче вам совершенно не важно, какие именно числа встретились в строке, важно только их количество.
Если программа читала символы и встретилась цифра, то она меняет стейт. И наоборот: если программа читала цифры и встретился иной символ, то число закончилось, нужно вернуть стейт обратно.
Итоговый ответ равен числу изменений стейта при обнаружении цифры. Для подсчёта можно использовать дополнительную переменную-аккумулятор.
Как будет тестироваться код
Вывод вашей программы будет сравниваться с эталонным.
